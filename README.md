## ⚡ QUICK START (30 MINUTES)

### 1. Install Git
```bash
Download: git-scm.com
Install with default settings
```

### 2. Configure Git
```bash
git config --global user.name "Mduduzi Msiza"
git config --global user.email "your-email@gmail.com"
```

### 3. Create GitHub Account
```
Visit: github.com/signup
Verify email
```

### 4. Create Repository
```
Visit: github.com/new
Name: finwise-app
Public: Yes
Add README: Yes
```

### 5. Clone Repository
```bash
git clone https://github.com/YOUR-USERNAME/finwise-app.git
cd finwise-app
```

### 6. Copy Your App
```
Copy finwise_complete.html to finwise-app folder
```

### 7. First Commit
```bash
git add .
git commit -m "Initial commit: Add FinWise app"
git push -u origin main
```

### 8. Deploy to GitHub Pages
```
Settings → Pages → Branch: main → Save
Wait 1-2 minutes
Visit: https://YOUR-USERNAME.github.io/finwise-app/
```

### 9. Rename for Direct Access (Optional)
```bash
mv finwise_complete.html index.html
git add index.html
git rm finwise_complete.html
git commit -m "Rename app to index.html"
git push
```

**Done! Your app is live!** ✅
